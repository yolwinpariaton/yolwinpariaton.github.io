# Automated Data Visualisation Portfolio

Portfolio and final project for PP434 Automated Data Visualisation for Policymaking at LSE.

Live Website: [yolwinpariaton.github.io](https://yolwinpariaton.github.io)

## Contents

Portfolio: Eight weekly tasks covering Vega-Lite charts, web scraping, APIs, maps, dashboards, and machine learning.

Project: UK Cost of Living Crisis - analyzing whether household incomes kept pace with inflation (2021-2024) through eight interactive visualizations using ONS, Ofgem, and government data.

## Technologies

- Visualization: Vega-Lite (HTML/JSON specifications as taught in class)
- Styling: CSS and JavaScript added to improve presentation and user experience
- Data Processing: Python (pandas, geopandas), APIs, web scraping
- Hosting: GitHub Pages

## Note

While the course focused on HTML and JSON code for visualizations, I added CSS styling and JavaScript for enhanced presentation and interactivity.
