The UK Cost of Living Crisis

An Economic Data Analysis

11.1% Peak Inflation (Oct 2022)
-2.5% Real Wage Growth (2022)
54% Energy Price Rise (2021-23)

1. Aims

This project examines the UK's cost of living crisis through a data-driven lens, focusing on the period from 2020 to 2024. The analysis seeks to quantify the crisis's impact across different dimensions: inflation dynamics, real wage erosion, regional disparities, and sectoral variations. By combining multiple data sources and visualization techniques, the project aims to provide a comprehensive picture of how rising costs have affected UK households, identify the most affected demographics and regions, and explore the underlying drivers of the crisis. The ultimate goal is to create an accessible, updateable resource that tracks the crisis's evolution and informs policy discussions.

2. Data Sources & Automation

The project utilizes multiple authoritative data sources, prioritizing APIs for automatic updates. Primary data comes from the Office for National Statistics (ONS) API, providing real-time access to inflation (CPIH, CPI), wage growth (AWE), and employment statistics. The Bank of England's Statistical Interactive Database supplies interest rate data and inflation expectations through their REST API. Energy prices are sourced from Ofgem's data portal using automated CSV downloads, while food prices utilize the ONS's average prices dataset.

For international comparisons, the project integrates World Bank and OECD APIs, enabling cross-country inflation and wage analysis. Regional data combines ONS regional statistics with the English Indices of Multiple Deprivation. All data pipelines are automated using Python scripts that fetch, clean, and transform data daily. The automation includes error handling for API failures and data validation checks. Raw data is stored in JSON format on GitHub, with transformation scripts documented in accompanying Jupyter notebooks, ensuring complete reproducibility.

3. Tools & Analytical Approach

The analytical pipeline employs Python for data processing, with pandas handling data manipulation and requests managing API calls. A key challenge was harmonizing different data frequencies‚Äîconverting weekly wage data to monthly for alignment with inflation series. This required developing custom interpolation functions that preserve seasonal patterns. The ONS API's pagination limits necessitated implementing recursive fetching for large datasets.

Visualization leverages Vega-Lite's declarative grammar, enabling responsive, interactive charts. Complex visualizations like the regional choropleth required merging ONS boundary data with economic indicators using GeoPandas. The energy price calculator presented unique challenges in modeling variable tariffs and government support schemes, solved by creating a parametric model that adjusts for household size and consumption patterns. All processing scripts are modularized for maintainability, with automated testing ensuring data quality. GitHub Actions runs daily updates, maintaining data freshness without manual intervention.

4. Conclusions

The analysis reveals a multifaceted crisis where energy prices acted as the initial catalyst, but broader supply chain disruptions and labor market tightness sustained inflationary pressures. Real wages fell by 2.5% at the crisis peak, with lower-income households experiencing effectively higher inflation due to spending patterns concentrated on essentials. Regional disparities are significant‚ÄîLondon faces highest absolute costs, but Northern regions suffer greater relative impact given lower median incomes.

The interactive visualizations enable users to explore their specific circumstances, making abstract economic data personally relevant. This project provides a foundation for ongoing monitoring, with automated updates ensuring continued relevance. Future work could expand sectoral analysis and incorporate micro-level household survey data for deeper demographic insights.
