Task 5: Accessing data: Scraper and API

I created one chart from a live API call and one chart from scraped Wikipedia data, then cleaned and reshaped the scraped data into tidy format.

Part A: API chart

This chart pulls UK inflation data directly from the World Bank Indicators API.

Functional description of the API call

  • Base URL: https://api.worldbank.org/v2
  • Country: /country/GBR
  • Indicator: /indicator/FP.CPI.TOTL.ZG
  • Query parameters: ?format=json&per_page=200
  • Final URL used in the chart: api.worldbank.org/.../FP.CPI.TOTL.ZG?format=json&per_page=200

Part B: Scraped chart

This chart uses emissions data scraped from Wikipedia, cleaned into tidy format, and visualises the five largest greenhouse gas emitters in 1970.

I scraped Wikipedia tables, handled inconsistent headers, reshaped to tidy format, and validated units before plotting the top emitters for a single year.
