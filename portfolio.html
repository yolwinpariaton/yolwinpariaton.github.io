Task 5: Live API chart

This chart pulls UK inflation data live from the World Bank Indicators API.

Functional description of the API call

  • Base URL: https://api.worldbank.org/v2
  • Country: /country/GBR (United Kingdom)
  • Indicator: /indicator/FP.CPI.TOTL.ZG (Inflation, consumer prices — annual %)
  • Query parameters: ?format=json&per_page=100 (JSON output; return up to 100 rows)
  • Final URL used in the chart: Link to API
